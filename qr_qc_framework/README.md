
# QR Degradation Framework

Install:

pip install opencv-python numpy scipy matplotlib pandas tqdm

Put QR images into `qr_codes/`.

Run:

python main.py

Outputs:
- output/results.csv
- output/summary.csv
- output/*.png plots

Artifacts:
- Gaussian Blur
- Motion Blur
- Gaussian Noise
- Exposure
- Contrast
- Haze
- Colorfulness
